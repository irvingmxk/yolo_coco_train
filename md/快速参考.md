# YOLO11n 快速参考

## 环境激活
```bash
conda activate yolo
```

## 训练模型
```bash
# 方式1：详细配置训练（推荐）
python train_yolo11n.py

# 方式2：简化版训练
python train.py
```

## 模型推理
```bash
python inference.py
```

## 数据集信息
- **类别**: bubble（气泡）
- **训练集**: 53 张图片
- **验证集**: 13 张图片
- **数据路径**: `/workspace/yolo/data/`

## 训练输出
训练完成后，模型保存在：
```
runs/train/yolo11n_bubble/weights/
├── best.pt   # 最佳模型（mAP最高）
└── last.pt   # 最新模型
```

## 训练配置
- **模型**: YOLO11n
- **训练轮数**: 100
- **批次大小**: 16
- **图像大小**: 640
- **优化器**: AdamW
- **学习率**: 0.01
- **早停**: patience=50

## 修改配置
编辑 `train_yolo11n.py` 中的 `CONFIG` 字典：
```python
CONFIG = {
    'epochs': 100,      # 修改训练轮数
    'batch': 16,        # 根据显存调整
    'device': 0,        # GPU编号或'cpu'
    # ...
}
```

## 批次大小建议
- **4GB 显存**: batch=8
- **8GB 显存**: batch=16
- **16GB+ 显存**: batch=32

## 常见问题

### 显存不足
```python
CONFIG['batch'] = 8  # 减小批次
```

### 使用CPU训练
```python
CONFIG['device'] = 'cpu'
```

### 恢复训练
```python
model = YOLO('runs/train/yolo11n_bubble/weights/last.pt')
model.train(resume=True)
```
